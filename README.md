# Fixed Income


## APP

Instalar todas depêndencias do projeto
```bash
$ npm install
```

Para executar
```bash
$ npm start
```

## API

Buildar aplicação
```bash
$ dotnet build
```
Executar aplicação

Os comandos devem ser executados na pasta raiz do projeto
```bash
$ dotnet run --project B3.FixedIncome\B3.FixedIncome.Api.csproj
```
Executar testes 
```bash
$ dotnet run --project B3.FixedIncome\B3.FixedIncome.Api.csproj
```
